# CineScope

CineScope is a simple movie search application built using JavaScript and an external movie API.

## Features

* Search movies by name
* View movie details like title, poster, and year
* Simple and clean interface

## Tech Stack

* HTML
* CSS
* JavaScript
* OMDb API

## How to Run

1. Download or clone the project
2. Open `index.html` in your browser

## API Setup

* Get an API key from OMDb API
* Add your API key in the JavaScript file

## Folder Structure

CineScope/

* index.html

* script.js
